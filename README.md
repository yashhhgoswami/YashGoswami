## üöÄ Getting Started

### 1. Clone & Install

```powershell
git clone https://github.com/yashhhgoswami/YashNewPortfolio.git
cd YashNewPortfolio
npm install
```

### 2. Run Dev Server

```powershell
npm run dev
```
Visit the printed local URL (default `http://localhost:3000`).

### 3. Lint

```powershell
npm run lint
```

### 4. Production Build

```powershell
npm run build
npm start
```
