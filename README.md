## Getting Started

Prerequisites:
- Node.js 18+ and npm (or Yarn/Pnpm)

Install dependencies and run the dev server:

```powershell
npm install
npm run dev
```

Build and start production server:

```powershell
npm run build
npm start
```

Lint:

```powershell
npm run lint
```
